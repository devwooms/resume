
## 경력사항

주요 업무 :
- Android, iOS 모바일 앱 개발 및 지속적인 유지보수와 추가 개발
- Window 앱 개발 및 지속적인 유지보수와 추가 개발
- Python 라이브러리 지원 및 지속적인 유지보수
- 자사 홈페이지 및 자사 자체몰 관리와 서버 유지보수
- 타사와의 협업 및 기술 지원  
     *(KT - Ai Codiny, 팀모노리스 - Codle, 서양네트웍스 - R.Robot)*

주요 성과 :
